# Аргос

Черновой пакет Аргоса, собранный по присланному тексту.

Города:
- Белогорье: посольство, турагентство, «Ярило», монастырь.
- Новоаргос: музей фотографии, Уделка, Снежный дворец.
- Мореград: площадь у Туманного кряжа, порт.

Главные улики:
- князь любит медовые сладости;
- Александра Романова выдает привычка делать всё левой рукой, потому что он стоит по правую руку князя;
- князю важны солнечные символы и свет за спиной;
- в Мореграде табу: перевёрнутый огонь и нарушение личного пространства.
