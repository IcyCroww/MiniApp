## Что даёт Элеон

Элеон собирает культурный слой саммита:

- настоящая картина в переговорной важнее случайной репродукции;
- правильный кофе и подача считываются как уважение;
- лилии для официальной встречи не подходят;
- рассадка и соседство за столом читаются как протокол.

## Маршрут v1

Старт открыт только в `Валеморе`.

Цепочки открытия:

1. `valemor.travel_agency` открывает `Сен-Альден`, `Беллерош`, `Монтелье`, `Сент-Лорен`.
2. `sen_alden.travel_agency` открывает `Вердаль`, `Лавьер`, `Арденваль`.
3. `bellerosh.travel_agency` открывает `Кастревиль`.

Закрытые города видны на мировой карте сразу, но до открытия показывают короткую подсказку.

## Города и механики

### Валемор

- `Угловое кафе «Le Горизонт»` -> `empty`
- `Туристическое агентство «Кофе и карты»` -> `empty` + открытие следующих городов
- `Министерство иностранных дел` -> `hotspot`
- `Музей современного искусства имени Жана Кокто` -> `hotspot`
- `Набережная у сувенирной лавки` -> `empty`

### Сен-Альден

- `Туристическое агентство «Мимы на прогулке»` -> `empty` + открытие следующих городов
- `Улица Ар-Нуво` -> `hotspot`
- `Цветочная лавка «Цветы и Этикет»` -> `caesar`, ответ `ЛИЛИЯ`

### Беллерош

- `Туристическое агентство «Дежавю»` -> `empty` + открытие `Кастревиля`
- `Мастерская тканей «Пряжа и Холст»` -> `choice`
- `Мастерская «Ларошет»` -> `empty`

### Монтелье

- `Туристическое агентство «Млечный путь»` -> `empty`
- `Кофейня «У Люси»` -> `empty`

### Вердаль

- `Туристическое агентство «Возвращение домой»` -> `empty`
- `Банкетный зал «Фарфор»` -> `choice`

### Пустышки

- `Сент-Лорен` -> мимы и пропавший кошелёк
- `Лавьер` -> случайная съёмочная группа
- `Арденваль` -> музыкант с виолончелью
- `Кастревиль` -> пекарский эпизод с круассанами
